# ===============================================
# 注記
# ===============================================
# 
# 【必須】
# - discord.py: Discord Bot の基盤
# - aiohttp: HTTP/HTTPS リクエスト、Web サーバー
# - websockets: WebSocket 接続（Wolfx/P2P）
# - pygame: MP3 再生
# - python-dotenv: 環境変数読み込み
#
# 【推奨】
# - psutil: メモリ監視（Web ダッシュボード使用時）
#
# 【オプション】
# - pytest / pytest-asyncio: テスト実行時
# - flake8 / black / isort: コード品質管理
# - opentelemetry-*: APM (Mackerel連携)。.env で APM_ENABLED=true にする場合のみ必要
#
# ===============================================
# インストール方法
# ===============================================
#
# すべてのパッケージをインストール:
#   pip install -r requirements.txt
#
# 必須パッケージのみインストール:
#   pip install discord.py aiohttp websockets pygame python-dotenv
#
# 推奨パッケージも含めてインストール:
#   pip install -r requirements.txt psutil
#
# 開発用パッケージも含めてインストール:
#   pip install -r requirements.txt pytest pytest-asyncio flake8 black isort
#
# APM (Mackerel連携) を有効化する場合のみ追加インストール:
#   pip install opentelemetry-api opentelemetry-sdk \
#       opentelemetry-exporter-otlp-proto-http \
#       opentelemetry-instrumentation-aiohttp-client
#   その後 .env で APM_ENABLED=true と APM_MACKEREL_API_KEY を設定
#
# ===============================================
# バージョン確認
# ===============================================
#
# インストール後、以下でバージョンを確認:
#   pip list | grep -E "discord|aiohttp|websockets|pygame|psutil"
#
# APM関連パッケージのバージョン確認:
#   pip list | grep -E "opentelemetry"
#
# ===============================================
